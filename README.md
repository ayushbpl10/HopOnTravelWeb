# HopON Travel – Website

Marketing website for the [HopON Travel](https://github.com/ayushbpl10/HopONTravel) mobile app.

## Tech Stack
- Pure HTML5, CSS3, JavaScript (no frameworks)
- Google Fonts (Outfit)
- Fully responsive & mobile-first

## Sections
- **Hero** – Full-screen banner with app download CTAs
- **Features** – 6 key features with animated cards
- **How It Works** – 4-step visual guide
- **App Showcase** – Phone mockups demonstrating the UI
- **For Vendors** – Trip organizer benefits & dashboard preview
- **Download** – App store CTAs
- **Footer** – Links, social, legal

## Local Development
Just open `index.html` in your browser — no build step needed.

## Deploy
Recommended: [GitHub Pages](https://pages.github.com/)
1. Go to repo Settings → Pages
2. Set source to `main` branch / root
3. Site will be live at `https://ayushbpl10.github.io/HopOnTravelWeb`
